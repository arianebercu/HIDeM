# HIDeM

<!-- badges: start -->

<!-- badges: end -->

The goal of HIDeM is to perform variable selection in an illness-death
model with interval-censored time to illness.

## Installation

You can install the development version of HIDeM from
[GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("arianebercu/HIDeM")
```

## Tutorials
